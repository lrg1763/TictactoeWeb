# Крестики-нолики (мультиплеер по ссылке)

Игра для двух игроков в браузере в реальном времени. Один игрок создаёт игру и копирует ссылку, второй переходит по ней и присоединяется к комнате.

## Стек

- **Backend:** Node.js, Express, WebSocket (ws)
- **Frontend:** HTML, CSS, JS (без фреймворков)
- **Шрифт:** [Manrope](https://fonts.google.com/specimen/Manrope) (Google Fonts)
- **PWA:** favicon, `site.webmanifest`, иконки для добавления на главный экран

## Запуск

```bash
npm install
npm start
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)

**Как играть:**

- **Создать игру** — создаётся комната, появляется ссылка для друга.
- **Присоединиться** — введите код комнаты из ссылки или откройте присланную ссылку.

## Деплой

Запустите на хостинге с Node.js (VPS, Railway, Render и т.п.):

```bash
node server.js
```

Для доступа по ссылке извне нужны **HTTPS** и **WSS** (WebSocket Secure).

Переменная окружения `PORT` задаёт порт (по умолчанию `3000`). Требуется Node.js 18+.

# Крестики-нолики (мультиплеер по ссылке)

Игра для двух игроков в браузере. Один создаёт игру и копирует ссылку, второй переходит по ней и присоединяется.

## Запуск

```bash
npm install
npm start
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)

- **Создать игру** — создаётся комната, появляется ссылка для друга.
- **Присоединиться** — введите код комнаты или откройте присланную ссылку.

## Деплой

Запустите `node server.js` на хостинге с поддержкой Node.js (VPS, Railway, Render и т.п.). Для работы по ссылке извне нужен HTTPS и WSS (WebSocket Secure).

Переменная окружения `PORT` задаёт порт (по умолчанию 3000).
